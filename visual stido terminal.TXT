$ git init - Git proqramini layihenin uzerine yukleme ve yeni izleme istemi yaratmaq
$ git status - faylim uzerinde aparilmis deyisiklikler

Her hansi xususi fayli izleme sistemine elaqe edilmesi:
$ git add .  - fayldaki butun fayllarin izlemeye aid edilmesi

$ git comit - m 'yeni fayllar' - gorulen islere oz komentini elave etmek. Yaddan cixmamasi ucun
$ git log - gorulen isleri gormek
$ git remote -v   - git-de elaqelendirilmis hesabi gormek

$ git push - u origin master - Local komputerden her hansi melumatin githubdaki hesaba online yuklenmesi. master yerine main olsaydi main yazmali idi

$ git pull origin master - u - Online github hesabdaki deyisilikleri localdakl terminalda gormek
